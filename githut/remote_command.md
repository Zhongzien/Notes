# git远程仓库
## 1.remote命令关联原创仓库
#### git remote add origin https://github.com/Zhongzien/xe_okhttp.git
## 2.push命令上传到git上
### <1>.首次上场
#### git push -u origin master
### <2>.首次上传以后，可以按如下方式使用
#### git push origin master 
## 3.clone命令从git远程库克隆到本地库(也可以理解为下载远程库到本地)
#### git clone https://github.com/Zhongzien/xe_okhttp.git
